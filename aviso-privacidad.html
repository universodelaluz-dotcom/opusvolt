Aviso de Privacidad

Ultima actualizacion: 11 de mayo de 2026.

En Opusvolt respetamos tu privacidad. Este aviso explica que datos podemos recopilar, para que los usamos y como puedes ejercer tus derechos.

1. Datos que podemos recopilar

  • Datos de contacto cuando nos escribes voluntariamente.
  • Datos tecnicos basicos de navegacion (por ejemplo, tipo de dispositivo o navegador).
  • Datos de uso agregados para mejorar la experiencia del sitio y nuestras apps.

2. Finalidades del tratamiento

  • Mejorar contenido, rendimiento y usabilidad del sitio.
  • Atender dudas, solicitudes o mensajes.
  • Analizar de forma estadistica el uso de nuestras paginas para optimizacion.

3. Base legal y consentimiento

Cuando sea necesario, solicitamos consentimiento para el uso de cookies y tecnologias similares. Puedes retirarlo en cualquier momento desde la configuracion de tu navegador.

4. Conservacion y seguridad

Aplicamos medidas razonables para proteger la informacion y conservarla solo durante el tiempo necesario para las finalidades descritas.

5. Derechos del usuario

Puedes solicitar acceso, rectificacion, actualizacion o eliminacion de tus datos personales conforme a la normativa aplicable en tu jurisdiccion.

6. Contacto

Para temas de privacidad, puedes contactarnos en opusvolt@gmail.com.

7. Cambios a este aviso

Podemos actualizar este aviso para reflejar mejoras del servicio o cambios normativos. Publicaremos aqui la version vigente con su fecha de actualizacion.
